Une variable est un petit espace de stockage (autrement dit une partie de la mémoire que l'on représente généralement par une case) sur l'ordinateur permettant d'enregistrer tout type de données, que ce soit une chaîne de caractères, une valeur numérique ou bien des structures plus complexes. vspace{10} Dans un premier temps, il est nécessaire de *{bold::déclarer une variable} : il s'agit de lui réserver un espace de stockage en mémoire, on utilise le mot clé code{var}. Une fois la variable déclarée, on peut commencer à y stocker des données. Chaque espace de stockage ainsi réservé (une variable) doit comporter un nom. Ce nom peut contenir autant de caractères souhaités. On peut utiliser les lettres de A à Z et les chiffres de 0 à 9 (non placés en première position du nom) ; l'underscore (_) et le dollar ($). Par exemple pour la variable de nom v1 la déclaration est la suivante : code{var v1;}. vspace{10} Après la déclaration, il est nécessaire d'*{bold::affecter} la variable c'est à dire stocker une valeur dans la case mémoire. On peut affecter une variable directement après sa déclaration ou après. Pour ce faire, il faut utiliser le symbole =. Pour affecter une variable de type chaîne de caractères (*{bold::string}), il est indispensable de placer la valeur entre ' ou ". Exemple : code{var text = "la valeur";}
