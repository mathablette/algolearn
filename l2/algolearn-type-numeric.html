Une variable de type numérique est une variable qui permet de stocker un nombre, par exemple : vspace{5} $\bullet$ La variable $n$ est affectée au nombre entier $11$ : code{var n = 11;} vspace{5} $\bullet$ La variable $x$ est affectée au nombre décimal $10,89$ : code{var x = 10.89;} vspace{5} $\bullet$ La variable $m$ est affectée à partir de la chaîne de caractère "4" : code{var m = parseInt("4");} vspace{5} $\bullet$ La variable $y$ est affectée à partir de la chaîne de caractère "8.02" : code{var m = parseFloat("8.02");}
Compléter suivant pour que l'affichage soit $9.5$ et $11$.
Les opérations sur les variables numériques sont multiples en voici les principales : vspace{5} $\bullet$ code{n++;} est équivalent à code{n = n + 1;} vspace{5} $\bullet$ code{n += 9;} est équivalent à code{n = n + 9;} vspace{5} $\bullet$ code{n *= 3;} est équivalent à code{n = n * 3;} vspace{5} $\bullet$ code{n /= 4;} est équivalent à code{n = n / 4;}
Compléter suivant pour que l'affichage soit $11$.
